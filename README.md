FocusPlan

O FocusPlan é uma aplicação web voltada para organização de estudos, tarefas e rotina acadêmica.

A proposta do projeto é ajudar estudantes a organizarem matérias, atividades, avaliações, prazos e prioridades em um único ambiente, tornando o planejamento mais claro e fácil de acompanhar.

O sistema foi desenvolvido como projeto da disciplina de Projeto Integrador II, utilizando práticas de Vibe Coding e ferramentas de Inteligência Artificial no processo de planejamento, documentação e desenvolvimento.

Objetivo do projeto

O principal objetivo do FocusPlan é auxiliar estudantes que possuem dificuldade para organizar sua rotina de estudos.

A aplicação permite centralizar informações importantes, como tarefas, disciplinas, prazos e prioridades, ajudando o usuário a visualizar melhor suas responsabilidades e acompanhar sua evolução.

Problema identificado

Muitos estudantes enfrentam dificuldades para:

* organizar matérias e atividades;
* acompanhar prazos de entrega;
* definir prioridades;
* planejar horários de estudo;
* visualizar tarefas pendentes e concluídas;
* manter uma rotina organizada.

O FocusPlan busca reduzir esses problemas por meio de uma interface simples, intuitiva e centralizada.

Funcionalidades

Entre as principais funcionalidades desenvolvidas no projeto estão:

* visualização de um painel com informações da rotina;
* cadastro e organização de tarefas;
* definição de prazos;
* classificação de atividades por prioridade;
* acompanhamento de tarefas pendentes e concluídas;
* organização das informações acadêmicas;
* navegação entre as principais áreas da aplicação;
* interface responsiva para computadores e dispositivos móveis.

A estrutura do projeto também está preparada para receber futuras funcionalidades baseadas em Inteligência Artificial, como:

* geração automática de planos de estudo;
* sugestão de prioridades;
* organização inteligente de tarefas;
* recomendação de horários;
* análise da carga de atividades.

Tecnologias utilizadas

O projeto foi desenvolvido com as seguintes tecnologias:

* Lovable — utilizado no processo de criação e evolução da aplicação;
* React — utilizado na construção da interface;
* TypeScript — utilizado para maior organização e segurança no código;
* Vite — utilizado como ferramenta de desenvolvimento e construção do projeto;
* Tailwind CSS — utilizado na estilização da interface;
* GitHub — utilizado para versionamento e armazenamento do código;
* Inteligência Artificial — utilizada como apoio no planejamento, documentação, geração e revisão do código.

Como executar o projeto

Para executar o FocusPlan localmente, é necessário ter o Node.js instalado.

Clone o repositório:

git clone LINK_DO_REPOSITORIO

Acesse a pasta do projeto:

cd focusplan

Instale as dependências:

npm install

Inicie a aplicação:

npm run dev

Depois, abra no navegador o endereço informado pelo terminal.

Também é possível utilizar o Bun:

bun install
bun run dev

Uso de Inteligência Artificial

A Inteligência Artificial foi utilizada como apoio durante diferentes etapas do projeto, incluindo:

* elaboração da proposta;
* organização dos requisitos;
* criação da documentação;
* desenvolvimento da interface;
* geração e revisão de componentes;
* identificação de possíveis melhorias;
* apoio na correção de erros.

Todo conteúdo gerado com IA foi revisado e adaptado de acordo com os objetivos do FocusPlan.

Arquitetura da aplicação

A aplicação segue uma estrutura web organizada em camadas.

O usuário acessa a interface pelo navegador. O front-end é responsável por exibir as páginas, receber as informações e executar as principais interações.

O projeto está preparado para se conectar a serviços de backend, banco de dados e recursos de Inteligência Artificial conforme a evolução da aplicação.

Fluxo simplificado:

Usuário
   ↓
Aplicação Web
   ↓
Front-end em React
   ↓
Serviços da aplicação
   ↓
Banco de dados e recursos de IA

Limitações atuais

Entre as melhorias futuras estão:

* autenticação completa de usuários;
* armazenamento permanente das tarefas;
* integração completa com banco de dados;
* geração de planejamento com IA;
* notificações de prazos;
* calendário de estudos;
* relatórios de produtividade;
* testes automatizados;
* maior controle de segurança e permissões.
